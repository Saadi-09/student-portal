# 🎓 Student Portal

A multi-page Student Web Portal built as part of the **CYS-463L Web Engineering Lab** sessional at the Department of Computer Science. The project demonstrates front-end fundamentals with a modern glassmorphism UI.

---

## 🌐 Live Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Landing page with hero section & feature cards |
| Registration | `registration.html` | Student sign-up form with JS validation |
| Dashboard | `dashboard.html` | Personalized welcome page with live date & time |
| Courses | `courses.html` | Course selection with custom checkboxes |
| Feedback | `feedback.html` | Feedback form with star rating & success state |

---

## ✨ Features

- **Multi-page navigation** — all pages connected through a sticky glassmorphism navbar
- **Form validation** — presence checks, email format, password match, numeric age type check
- **Bootstrap 5** — responsive grid, navbar, and layout utilities
- **Custom JavaScript** — live date/time, localStorage for name persistence, dynamic UI states
- **Glassmorphism UI** — `backdrop-filter` blur, radial gradient mesh background, vibrant accent colors
- **HCI-aware design** — hover microinteractions, focus states, error highlighting, intuitive star rating

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Custom Properties, Glassmorphism, Animations)
- Vanilla JavaScript (ES6)
- Bootstrap 5.3
